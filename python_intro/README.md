# Practice_coding

## python_intro
### 0_intro 
    Hello world! 및 기본 print문

    변수 사용을 통한 예제 학습

### 6_return
    return문, while문, scope(범위) 및 constant(상수) 관련 개념 및 예제

### 7_if_elif
    조건문 if와 elif의 개념, 구조 및 예제

### 12_module
    모듈(module)과 스탠다드 라이브러리(standard library)의 개념, 적용 및 예제

### 13_input
    input 함수의 개념과 적용, 예제

### 14_read_write_strip_spilt
